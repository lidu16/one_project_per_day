# Food Website

A simple static website showcasing food items with a hero section and product listings.

## Description

This project is a basic HTML and CSS webpage for a food service, featuring a hero image with a postcode input form and a grid of food products.

## Features

- Hero section with background image and call-to-action form
- Responsive product grid displaying food items
- Mobile-friendly design with media queries
- Custom fonts (Montserrat) from Google Fonts
